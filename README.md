# iris-dataset-visualization
Iris Dataset Exploration and Visualization

This project focuses on Exploratory Data Analysis (EDA) and data visualization using the famous Iris Flower Dataset. The dataset is loaded directly from the Seaborn library and analyzed to understand its structure, statistical properties, class distribution, feature relationships, and potential outliers.

Various visualization techniques are used to gain insights into the dataset, including scatter plots, histograms, box plots, and pair plots. These visualizations help identify patterns, correlations between features, species separation, and distribution characteristics.

The project demonstrates fundamental data analysis skills using Python libraries such as Pandas, Seaborn, and Matplotlib.

Features
Load the Iris dataset directly using Seaborn
Perform dataset inspection and exploration
Display dataset shape, columns, and summary statistics
Analyze species distribution
Visualize feature relationships using scatter plots
Examine feature distributions using histograms
Detect outliers using box plots
Generate pair plots for comprehensive feature comparison
Save all visualizations as image files
Technologies Used
Python
Pandas
Seaborn
Matplotlib
